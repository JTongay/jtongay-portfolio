Pi-TV

This was a very fun project. I had gotten a Raspberry Pi 3 for my birthday and wanted to do some sort of project with it for class. The guidelines for the project were very simple, create something outside of the box. I already had a few CRUD apps in my pocket, so this project got me very excited. My fellow class-mate and I decided to make a smart TV because we knew we wanted to incorporate a mobile app with project. The challenge for this project was to have it completed in one week. My class-mate and I took to the Agile Methodology to get this project completed within the time frame. Our MVP (minumumal viable product) was to get it to play a YouTube video. We also decided to document our entire process with a blog to give people a better idea our thoughts throughout the project. After a few internet problems in campus, we managed to get our MVP completed within the week.

My Contribution: The entire Front-End in Ionic.

Digital Rolodex

The Assignment: Work in a team of 6 to create a Full-Stack and full CRUD application using any API in 1 week.

The Challenge: It was everyones first time working in a group of this size, so the Git workflow was new.

My Contributions: The ERD for everyone to be on the same page with tables, columns, and relationships. The migrations and seeds for development and testing databases. The tags feature where you can drag and drop descriptive tags for adding a new contact.

The Result: A full CRUD digital rolodex to keep all of a users contacts information in case the user loses business cards. The user can sign-up/log-in, create new contacts by either filling out the form or uploading a picture of a business card to fill out the form for you. The user can assign tags to a contact to describe who that person is.
