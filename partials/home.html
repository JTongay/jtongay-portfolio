Pi-TV

The Assignment: Create something creative that is not a CRUD app in 1 week.

The Challenges: The most difficult part was to get the Ionic App to communicate with the Raspberry Pi. We had issues with the internet at campus, so we only really had 2 days of actual code written.

My Contribution: The entire Front-End in Ionic.

The Result: A fully functioning Raspberry Pi Chromecast-like device that will play YouTube videos controlled by the app. The video is not streamed, but downloaded to the device and has some issues with playing music videos.

Digital Rolodex

The Assignment: Work in a team of 6 to create a Full-Stack and full CRUD application using any API in 1 week.

The Challenges: It was everyones first time working in a group of this size, so the Git workflow was new.

My Contributions: The ERD for everyone to be on the same page with tables, columns, and relationships. The migrations and seeds for development and testing databases. The tags feature where you can drag and drop descriptive tags for adding a new contact.

The Result: A full CRUD digital rolodex to keep all of a users contacts information in case the user loses business cards. The user can sign-up/log-in, create new contacts by either filling out the form or uploading a picture of a business card to fill out the form for you. The user can assign tags to a contact to describe who that person is.

Pokemon Stay!

The Assignment: Make your first website using any API of your choice in 1 week.

The Challenges: Getting used to reading documentation for different API's and other JavaScript libraries. Also, the pathing for each pokemon in explore mode.

The Process: I read documentation like crazy on each API I used, Bootstrap, and Paperjs. Having my instructors as mentors was my first experience doing pair-programming and was a fantastic source of help.

The Result: An encyclopedia of Pokemon knowledge broken down by each generation and a related YouTube video of a selected pokemon. Also in browse mode, you can move your character around to "catch" a pokemon coming at you to see more information about it.
